## 1. Line Notify 介紹
> Line Notify官方介紹： https://notify-bot.line.me/zh_TW/  
> Line Notify API Document: https://notify-bot.line.me/doc/en/

簡單來說，Line Notify是由Line官方提供的訊息推播服務，可以單方面的傳送訊息給使用者。雖然不能與使用者對話，但是Notify功能是免費的。
